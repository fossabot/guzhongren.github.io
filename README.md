## 生成模板文章

```zsh
$ # make post="[article_name]"
$ make post="article"
```

## 预览

```zsh
$ make pre
```

## 提交修改

```zsh
$ # make commit message="提交信息"
$ make commit message="feat(post): 发布新文章"
```

## 发布

```zsh
$ make pub
```
